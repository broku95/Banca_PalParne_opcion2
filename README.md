# Proyecto-Banca-PalParne

Este proyecto es una aplicación sencilla hecha en Python que simula un banco básico.  
Es un trabajo para las clases de Python y sirve para practicar lo que hemos visto en clase como funciones, módulos, condicionales y bucles.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Con este programa puedes:

- Ver el saldo que tienes
- Ingresar dinero
- Retirar dinero
